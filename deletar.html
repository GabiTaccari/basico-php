Teoria :

A opção deletar no php, através de um formulário vai apagar o dado da tabela que for definida no php;

Sintaxe formulario html :

< form method="post" action="excluir.php">
< center>
< label>Digite o nome para excluir: < /label>
< textarea value="pesquisa" name="pesquisa" required>< /textarea>< p>
< input type="submit" name="Pesquisar" value="Excluir">
< input type="reset" name="Limpar" value="Limpar">
< /center>
< /form>

Sintaxe Deletar php:

< ?php
include_once('conexao.php');
$nome=$_POST["pesquisa"];
//executa a query com base na conexão
$query = mysqli_query($conexao, "DELETE FROM tabela WHERE nome_coluna = '{$nome}'");
//verifica se a query esta ok
if(!$query)
{
die('Query Inválida' . mysqli_error($conexao)); // mostra o erro
}
echo"Dados excluídos com sucesso";
//finaliza a conexao
mysqli_close($conexao);
?>
