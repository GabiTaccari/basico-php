Definição Array :

  Um array no PHP é na verdade um mapa ordenado. Um mapa é um tipo que relaciona valores a chaves. Este tipo é otimizado para várias usos diferentes: ele pode ser tratado como um array, uma lista (vetor), hashtable (que é uma implementação de mapa), dicionário, coleção, pilha, fila e provavelmente mais. Assim como existe a possibilidade dos valores do array serem outros arrays, árvores e arrays multidimensionais.

Sintaxe do Array:

Exemplo 1:
<?php
  $lista = array('conteudo1', 'conteudo2', 'conteudo3');
?>

   Neste primeiro exemplo da sintaxe, é definido todos os três elementos na forma mais simples possível, separados apenas por vírgula por vírgula.



Sintaxe do Array:

Exemplo 2:
<?php
  $lista = array();
  $lista[0] = 'conteudo1';
  $lista[1] = 'conteudo2';
  $lista[2] = 'conteudo3';
?>

  Já no segundo exemplo definiu-se (antes) que $lista será um array, e logo após, definiu-se seus três elementos, ptendo um número entre colchetes agora: $lista[1], que significa o índice do array, a posição do elemento. O índice do elemento pode ser definido como numérico ou textual (string)

Sintaxe do Array:

Exemplo 3:
<?php
$lista = array();
$lista[] = 'conteudo1';
$lista[] = 'conteudo2';
$lista[] = 'conteudo3';
?>

  No terceiro exemplo, faz-se a mesma coisa que do segundo, só que omite-se os índices dos elementos, sendo assim, o PHP irá colocar cada elemento no fim do array, começando com a posição 0 (primeiro elemento) e crescendo a cada elemento.

Índices Textuais (Strings) - Sintaxe Array :

Também é possível definir índices (também chamados de keys ou chaves) como strings, segue um exemplo de uso:

<?php
  // Definição longa
  $conjunto = array();
  $conjunto['chave0'] = 'elemento1';
  $conjunto['chave1'] = 'elemento2';
  $conjunto['chave2'] = 'elemento3';

  // Definição simples
  $conjunto = array('chave0' => 'elemento1', 'chave1' => 'elemento2', 'chave2' => 'elemento3');
?>

Exibindo e localizando elementos dos arrays

De maneira simplificada, para exibir um elemento basta seguir a mesma sintaxe da declaração longa:

<?php

  //Declaração longa:
  $conjunto = array();
  $conjunto['chave0'] = 'elemento1';
  $conjunto['chave1'] = 'elemento2';
  $conjunto['chave2'] = 'elemento3';
  echo "O elemento do segundo vetor é: " . $conjunto['chave1'];

  // Lista
  $lista = array('conteudo1', 'conteudo2', 'conteudo3');
  echo "O segundo item da lista é: " . $lista[2];
?>

Arrays multi-dimensionais (matriz)

De maneira simplificada, pode-se definir um array multi-dimensional onde um ou mais elementos sejam arrays. Nesse caso o array não pode mais ser chamado de lista, e deve ser chamado de vetor ou array mesmo. Veja um exemplo básico de definição e exibição:

<?php

  // Carro
  $carro = array();
  $carro['cores'] = array('Vermelho', 'Branco', 'Cinza');
  $carro['modelo'] = 'CrossFox';
  $carro['fabricante'] = 'Volkswagen';
  echo "A terceira cor do meu " . $carro['modelo'] . " é: " . $carro['cores'][2];
  // Resultado: A terceira cor do meu CrossFox é: Cinza
?>
