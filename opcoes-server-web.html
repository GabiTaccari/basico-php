Passo 1.

Clique com o botão direito no ícone do WampServer, ele fica ao lado do relógio do Windows na “Área de Notificação”. No menu que aparece, clique na opção “Language” e em seguida no item “Português”;

Passo 2.

Clique com o botão esquerdo sobre o mesmo ícone e serão exibidas todas as opções para o servidor do programa, como parar, reiniciar e iniciar os serviços (php, mysql, apache).

Acessos:

     Para acessar o servidor web local: opção local host do menu.


     Para acessar o gerenciador do banco de dados Mysql, clique na opção “phpMyAdmin”


     Para acessar o diretório dos arquivos públicos do site, clique na opção “diretório www”.


Observação:

Para executar o wamp com total funcionalidade, é necessário parar todos os serviços relacionados a banco de dados que estiverem rodando na máquina.
