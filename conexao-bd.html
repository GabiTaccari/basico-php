Teoria:

É necessário a criação de um arquivo de extensão .php para a realização da conexão com o banco de dados. É um arquivo de extrema importância, pois para os demais arquivos php que tiverem interação com o banco de dados será necessária a execução da chamada do arquivo que fará a conexão com o banco.

Código:

<?php

  //Define as variáveis necessárias para realizar a conexão
  $host = "localhost";
  $user = "root";
  $pass = "";
  $banco = "projetophp";

  //Verifica se a conexão é realizada com sucesso ou se ocorre algum erro ao conectar
  $conexao = mysqli_connect($host, $user, $pass, $banco)
    or die("Problemas com a conexao do banco de dados");
?>
