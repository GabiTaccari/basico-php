Teoria :

A opção alterar no php, através de um formulário vai alterar o dado da tabela que for definida no php;

Sintaxe formulario html :

< form method="post" action="alterar.php">
< center>
< label>Digite o nome para alteração: < /label>
< textarea value="pesquisa" name="pesquisa" required>< /textarea>< p>
< label>Digite o dado para alteração: < /label>
< textarea value="pesquisa" name="pesquisa" required>< /textarea>< p>
//pode-se criar quantos campos forem necessarios para alteração dos campos da tabela do banco
< input type="submit" name="Pesquisar" value="Alterar">
< input type="reset" name="Limpar" value="Limpar">
< /center>
< /form>

Sintaxe Alterar php:

< ?php
include_once('conexao.php');
//pode-se criar a quantidade de variaveis necessarias relacionadas aos campos, ou seja, a quantidade de campos existentes na tabela do BD
$var1 = $_POST['v1'];
$var2 = $_POST['v2'];
$var3 = $_POST['v3'];
//executa a query com base na conexão
$query = mysqli_query($conexao, "UPDATE nome_tabela set coluna1 = '$var1', coluna2 = '$var2', coluna3 = '$var3'");
//verifica se a query esta ok
if(!$query)
{
die('Query Inválida' . mysqli_error($conexao)); // mostra o erro
}
echo"Dados alterados com sucesso";
//finaliza a conexao
mysqli_close($conexao);
?>
