Segunda forma:

Terceira forma:

<script language="php">

Quarta forma:

%>

Comando echo:

    É a instrução que envia para a página de saída, em HTML, as informações processadas em PHP, podendo ser texto, números ou variáveis. Essas informações, na maioria das vezes, são mescladas com os comandos HTML.


Dentro do comando echo, as aspas utilizadas para os comando são simples, e não duplas.


Sintaxe:


<?php
         echo "O meu texto tem 'aspas' ";
?>
    

Concatenando:

A concatenação é realizada com um ponto ( . ), com a seguinte sintaxe:


<?php
// Cria as variáveis com o valores necessários
    $meunome = 'Luiz Santos';
    $minhaidade = 27;
// Exibe uma frase com as variáveis concatenadas
    echo 'Olá, meu nome é ' . $meunome . ' e minha idade é ' . $minhaidade;
?>



Tipos de variáveis:


As variáveis em PHP não precisam ser declaradas. Quando atribuímos algum valor para elas, o tipo é automaticamente reconhecido.


Tipo Descrição
Inteiro Número inteiro (...18, 1, 0, -1, -54...)
Ponto Flutuantes Números reais (...3.14, -5.48...)
String Cadeia de caracteres ("João da Silva")
Array Coleções
Objeto Instancia de Classes
Booleano TRUE ou FALSE
Null Sem valor atribuido

Variáveis


Representam um espaço na memória onde um valor está armazenado

No PHP começam pelo símbolo cifrão ( $ )

São case sensitive (diferem maisculas de minusculas

Podem ter letras, underline e numeros

Não podem ter espaços e comecar por número

Não precisa declarar o tipo antes, basta utilizá-las
